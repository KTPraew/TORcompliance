## Design Context

### Users
พนักงาน IT เอกชน — ทีม IT ของบริษัทที่รับพัฒนาระบบให้ภาครัฐ ใช้งานในออฟฟิศกลางวัน บนหน้าจอ desktop เป็นหลัก ต้องการตรวจสอบว่าเว็บไซต์ที่พัฒนาผ่านมาตรฐาน WCAG/TWCAG/Policy ก่อน submit งาน ผู้ใช้มีความรู้ด้าน IT สูง ต้องการ efficiency และ clarity มากกว่า handholding

### Brand Personality
**Bold · Precise · Authoritative**  
เครื่องมือวิชาชีพที่สร้างความมั่นใจให้ผู้ใช้ว่า output ถูกต้องและเชื่อถือได้ ไม่ใช่แค่ checker ธรรมดา แต่เป็น compliance expert ในมือ

### Aesthetic Direction
Bold & Modern — ทันสมัย มั่นใจ แต่ยังคงความ professional  
ไม่เป็น government-software ที่ล้าสมัย ไม่เป็น generic SaaS ที่จืดชืด  
อ้างอิงที่ดี: Linear, Vercel Dashboard, Raycast — dense, dark sidebar, confident typography  
ต้องไม่เป็น: generic light-mode B2B, cyan-on-dark AI aesthetic, gradient everything  
Theme: รองรับทั้ง light และ dark mode

### Platform
Web app (Next.js 14) — desktop primary, tablet secondary, mobile not primary use case  
Browser: modern Chrome/Edge ในบริบท enterprise  
ไม่มี React Native / mobile app

### Design Principles
1. **Data first** — ข้อมูล compliance score และ checklist items ต้องเห็นได้ชัดทันที
2. **Confidence by density** — layout ที่ dense และ information-rich สื่อถึงความเป็น professional tool
3. **Accessible UI for an accessibility product** — ระบบที่ตรวจสอบ WCAG ต้องผ่าน WCAG AA เอง
4. **Consistent tokens** — brand colors ต้องใช้ผ่าน CSS variables ทุกที่ ไม่ hardcode ใน JSX

### Brand Overrides
- ไม่มี font override — Sarabun เป็นตัวเลือกใหม่ ไม่ได้อยู่ใน reflex ban list
- ไม่มี pattern override ที่อนุญาตพิเศษ

## Design Tokens

### Colors
- `--primary`: #4361ee (oklch approx 55% 0.18 264)
- `--primary-light`: #748ffc
- `--primary-dark`: #2d44c5
- `--sidebar`: #1e3161 (navy dark)
- `--surface`: #f0f5ff (light blue-white)
- `--accent-bg`: #eef2ff
- `--accent-border`: #c7d2fe
- Status green: emerald-500 / #10b981
- Status red: red-500 / #ef4444
- Status amber: amber-500 / #f59e0b

### Typography
- **Font**: Sarabun (Thai + Latin, Google Fonts)
- **Weights**: 300, 400, 500, 600, 700
- **Scale used** (px): 10 / 11 / 12 / 14 / 16 / 18 / 20 / 24 / 32
- **Variable**: `--font-sans`

### Spacing
Tailwind default 4pt scale: 4 / 8 / 12 / 16 / 20 / 24 / 28 / 32 / 40 / 48 / 64 / 96

### Border Radii
- `rounded-lg`: 0.75rem (12px) — via `--radius`
- `rounded-xl`: custom 12px
- `rounded-2xl`: 1rem (16px)
- `rounded-3xl`: 1.5rem (24px)

### Shadows
- Card: `0 1px 4px rgba(67,97,238,0.04), 0 4px 20px rgba(67,97,238,0.06)`
- Card hover: `0 4px 24px rgba(67,97,238,0.12), 0 1px 4px rgba(67,97,238,0.06)`
- Modal: `0 8px 40px rgba(30,49,97,0.18)`

### Breakpoints
Tailwind defaults: sm 640 / md 768 / lg 1024 / xl 1280 / 2xl 1536
